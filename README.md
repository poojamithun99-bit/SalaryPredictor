# 💼 SalaryPredict AI Enterprise

A high-fidelity salary prediction system built with **Python**, **Streamlit**, and **Scikit-Learn**. 

## ✨ Features
* **Machine Learning:** Uses Linear Regression to predict compensation based on tenure.
* **Interactive UI:** Minimalist Sage Green and Black theme with real-time Plotly charts.
* **Global Ready:** Multi-currency support (USD, INR, EUR) and role-based multipliers.

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com/poojamithun99-bit/SalaryPredictor.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
